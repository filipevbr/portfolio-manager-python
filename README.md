# Portfolio Manager (CLI)

A command-line interface (CLI) application developed in Python to manage personal projects. This project demonstrates core programming concepts such as **CRUD operations**, **Modular Architecture**, **JSON Persistence**, and **Clean Code** principles.

## 🚀 Features

- **ADD:** Register new projects with a name and initial status.
- **LIST:** View all projects with status (Pending/Completed) and history logs.
- **UPDATE:** Modify project names or toggle status.
- **DELETE:** Remove projects safely (with confirmation).
- **PERSISTENCE:** Data is automatically saved to and loaded from `portfolio.json`.
- **ENGLISH CODEBASE:** All variables, functions, and interfaces are written in technical English.

## 🛠️ Technologies

- **Language:** Python 3.x
- **Data Storage:** JSON (Local File)
- **Architecture:** Modular (Separation between Logic and Utilities)

## 📂 File Structure

- `main.py`: Contains the core business logic and menu navigation.
- `utils.py`: Contains utility functions for input validation and UI/UX.
- `portfolio.json`: Automatically generated file for data storage.
- `ai_log.md`: A log of the development process assisted by AI.
